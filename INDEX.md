## 🎯 Quick Reference

### To Change Phone Number
1. Open `index.html`
2. Find: `9876543210`
3. Replace with: Your phone number
4. Also update in `script.js`

### To Update Address
1. Open `index.html`
2. Find: "Siwan" or "Bhikhabandh"
3. Update in all sections
4. Update in Footer

### To Change Colors
1. Open `styles.css`
2. Find: `:root { ... }`
3. Modify: `--primary-red: #DC143C;`
4. Save and refresh browser

### To Add Products
1. Open `index.html`
2. Find: `<!-- Products Section -->`
3. Copy a product card
4. Modify name, price, image

### To Update Prices
1. Open `index.html`
2. Find: `<!-- Today's Price List -->`
3. Edit the table rows

## 📝 File Modification Guide

### Safe to Edit:
- Text content in `index.html`
- Prices in tables
- Colors in `styles.css` :root
- Phone numbers in all files
- Product descriptions

### Be Careful With:
- HTML structure (may break layout)
- CSS class names (must match HTML)
- JavaScript functions (affects features)
- External links and CDN URLs

### Never Delete:
- Opening/closing HTML tags
- CSS media queries
- JavaScript event listeners
- Meta tags
